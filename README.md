# Firewall-Traffic-Simulator
Developed a menu-based Firewall Traffic Simulator in Java that applies core Object-Oriented Programming (OOP) principles to monitor, filter, and manage network traffic records efficiently.

My Key Technical Contributions:
* Advanced OOP Architecture: Designed the specialized WebTraffic class, actively utilizing Inheritance, Encapsulation, and Polymorphism to structure complex network data.
* Automated Security Filtering: Engineered an automated logic system that parses incoming traffic and automatically blocks malicious URLs or unsafe ports using the ActionStatus Enum.
* Rate Limiting & Threat Detection: Developed a duplicate detection algorithm utilizing Safe Object Downcasting (instanceof) to identify and flag redundant access attempts from the same IP.
* Data Integrity: Implemented strict input validation mechanisms to ensure robust system stability against invalid data entries.
